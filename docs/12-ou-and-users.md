# Organizational Units and Users

## Creating OUs

1. Open **Active Directory Users and Computers (ADUC)**.
2. Right-click `BESTIU.LOCAL` → New → Organizational Unit. 
3. Create the following OUs:
   - `Users`
   - `Computers` 
4. Under `Users`, create:
   - `Admins`
   - `Employees`
   - `Interns`
   - `HR` 

## Creating Users

1. Right-click an OU (e.g., `Admins`) → New → User. 
2. Provide user information such as `IT_Admin`, set logon name, and assign a strong password. 
3. Repeat for other user accounts (employees, interns, HR Manager) according to the designed structure. 
4. Verify the OU hierarchy and user placement in ADUC.
